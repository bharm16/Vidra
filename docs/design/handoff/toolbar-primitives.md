# The toolbar

The control strip attached to the input. Its contents change with the workflow moment.

## Contains

- **Primary action** — the one next step. Changes per moment:
  - empty → submit
  - writing / painting / moving → in-progress, non-advancing
  - picture, words unchanged → make it move
  - picture, words edited → remake the picture
  - clip → keep (carries the subscription offer for non-subscribers)
  - kept → new clip
- **Secondary action** — optional, present only some moments:
  - picture → re-roll the picture (same words)
  - picture, words edited → make it move
  - clip → re-roll the motion
- **Settings** — two inline selectors on the input: aspect ratio and clip duration.
